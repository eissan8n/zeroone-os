## 📖 نظرة عامة

**ZeroOne OS** هو "سوبر أب" عربي متكامل، مبني على مفهوم **الجيل الثاني من البرمجة**. يجمع بين واجهة أوامر نصية ذكية وواجهة مستخدم رسومية، ليكون مركز التحكم الوحيد الذي تحتاجه لإدارة حياتك الرقمية.

> 🏆 **إنجاز قياسي:** تم تخطيط وتصميم هذا النظام بالكامل في **48 ساعة فقط** باستخدام تقنيات الذكاء الاصطناعي (DeepSeek AI).

## 📋 الأوامر المدعومة

| الأمر | الوصف | مثال |
|-------|-------|------|
| `محفظتي` أو `رصيدي` | عرض الرصيد الحالي | `محفظتي` |
| `ضيف ملاحظة: [نص]` | إضافة ملاحظة جديدة | `ضيف ملاحظة: فكرة المشروع` |
| `ضيف مهمة: [نص]` | إضافة مهمة جديدة | `ضيف مهمة: كتابة تقرير` |
| `احسب [معادلة]` | حساب معادلة رياضية | `احسب 5+5` |
| `احسب بالفارسي [معادلة]` | حساب مع إظهار الناتج بالفارسي | `احسب بالفارسي 7+8` |
| `تتبع شحنتي` | محاكاة تتبع شحنة | `تتبع شحنتي` |
| `ساعدني` | عرض جميع الأوامر | `ساعدني` |

## 🛠️ التقنيات المستخدمة

- **Frontend:** React 18 + TypeScript
- **Styling:** CSS3 (Glassmorphism + Flexbox)
- **State:** React Hooks (useState, useEffect)
- **Storage:** Browser localStorage
- **Fonts:** Cairo (Google Fonts)
- **Deployment:** Vercel
