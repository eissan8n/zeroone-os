## ✨ المميزات

- 🧠 واجهة أوامر عربية ذكية
- 💰 محفظة مالية (26 وسيلة دفع)
- 📝 إدارة الملاحظات والمهام
- 🧮 حاسبة بالأرقام الفارسية
- 📦 تكامل مع Amazon و Alibaba
- 🤖 منسق ذكاء اصطناعي

## 📋 الأوامر

| الأمر | الوصف |
|-------|-------|
| `محفظتي` | عرض الرصيد |
| `ضيف ملاحظة: [نص]` | إضافة ملاحظة |
| `ضيف مهمة: [نص]` | إضافة مهمة |
| `احسب [معادلة]` | حساب |
| `احسب بالفارسي [معادلة]` | حساب فارسي |
| `تتبع شحنتي` | تتبع شحنة |
| `ساعدني` | عرض الأوامر |
