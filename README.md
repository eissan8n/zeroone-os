## 📖 نظرة عامة

**ZeroOne OS** هو "سوبر أب" عربي متكامل، مبني على مفهوم **الجيل الثاني من البرمجة**. يجمع بين واجهة أوامر نصية ذكية وواجهة مستخدم رسومية ثلاثية الأبعاد، ليكون مركز التحكم الوحيد الذي تحتاجه لإدارة حياتك الرقمية.

> 🏆 **إنجاز قياسي:** تم تخطيط وتصميم هذا النظام بالكامل في **48 ساعة فقط** باستخدام تقنيات الذكاء الاصطناعي (DeepSeek AI)، مقارنة بـ 3-6 أشهر بالطرق التقليدية.

## ✨ المميزات الرئيسية

### 🧠 واجهة أوامر ذكية (Command Bar)
- كتابة أوامر باللغة العربية الفصحى أو العامية.
- فهم اللهجات المختلفة (ضيف/أضف، وريني/شوف).
- تنفيذ فوري مع تغذية راجعة في سجل الأوامر.

### 💰 المحفظة المالية الموحدة (Super Wallet)
- دعم **26 وسيلة دفع** (20 بنكاً مصرياً + 6 محافظ رقمية).
- تتبع الرصيد والمعاملات.
- أوامر: `محفظتي`، `رصيدي`.

### 📝 إدارة المهام والملاحظات
- إضافة ملاحظات ومهام عبر الأوامر النصية.
- حفظ تلقائي في المتصفح (LocalStorage).
- أوامر: `ضيف ملاحظة: [نص]`، `ضيف مهمة: [نص]`.

### 🧮 حاسبة ذكية (Smart Calculator)
- دعم العمليات الحسابية المعقدة.
- **تحويل النتائج إلى الأرقام الفارسية القديمة** (صفر، يك، دو، ثه، جهار...).
- أوامر: `احسب [معادلة]`، `احسب بالفارسي [معادلة]`.

### 🌍 مراكز التجارة العالمية (Global Hubs)
- **Amazon Hub**: تتبع الشحنات، إدارة FBA، والدفع عبر Amazon Pay.
- **Alibaba Hub**: البحث عن المنتجات والموردين (B2B)، وتكامل مع ذكاء Accio AI.

### 🤖 منسق الذكاء الاصطناعي (AI Orchestrator)
- دعم منصات متعددة: Claude، ChatGPT، Gemini، Groq.
- تنفيذ أوامر مثل: `اسأل Groq: ما هو الذكاء الاصطناعي؟`

### 🎨 تصميم عصري (Sky Blue Glassmorphism)
- واجهة زجاجية بتأثير `backdrop-filter`.
- دعم كامل للغة العربية (RTL).
- تجاوب تام مع جميع أحجام الشاشات (Mobile-First).

## 📋 الأوامر المدعومة (Quick Commands)

| الأمر | الوصف |
| :--- | :--- |
| `محفظتي` أو `رصيدي` | عرض الرصيد الحالي للمحفظة المالية. |
| `ضيف ملاحظة: [النص]` | إضافة ملاحظة جديدة إلى لوحة الملاحظات. |
| `ضيف مهمة: [النص]` | إضافة مهمة جديدة إلى لوحة المهام. |
| `احسب [معادلة]` | إجراء عملية حسابية (مثال: `احسب 5+5`). |
| `احسب بالفارسي [معادلة]` | حساب المعادلة وعرض الناتج بالأرقام الفارسية. |
| `تتبع شحنتي` | محاكاة تتبع شحنة أمازون (حالة عشوائية). |
| `ساعدني` | عرض قائمة بجميع الأوامر المتاحة. |
